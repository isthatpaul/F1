# 🏎️ F1 2026 Drivers' Championship Prediction

A machine learning project that predicts the 2026 Formula 1 Drivers' Championship using historical race data from FastF1 and linear regression.

## 📋 Project Overview

**Goal:** Predict which driver will win the 2026 F1 Drivers' Championship based on historical performance data.

**Approach:**

- Load historical F1 data (2023-2025) using FastF1
- Engineer driver performance features
- Train linear regression model
- Predict 2026 championship points and winner

**Tech Stack:**

- Python
- FastF1 (F1 data API)
- scikit-learn (machine learning)
- pandas, numpy (data processing)
- matplotlib, seaborn, plotly (visualization)
